# Product Overview

## Purpose

This project is a Financial Dashboard that visualizes financial metrics using a React frontend and a FastAPI backend.

The application displays key financial indicators such as income, expenses, profit trends, category summaries, comparisons, alerts, and business metrics through charts and KPI cards.

## Main Features

- Financial KPI dashboard
- Monthly income and expense visualization
- Profit trend analysis
- Financial category summaries
- Business-to-Business (B2B) metrics
- Business-to-Consumer (B2C) metrics
- Financial comparison reports
- Financial alerts
- REST API for financial data

## Target Users

Developers learning how to understand an existing codebase and build project context before making changes.

## Architecture

- React frontend
- FastAPI backend
- Docker Compose local development
- Mock financial data generation
- REST API communication
